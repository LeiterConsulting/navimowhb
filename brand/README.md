# Branding Assets

This directory contains the reusable Navimow for Homebridge artwork set.

Files:

- `navimow-square.svg`: square brand mark for README, issue submissions, and icon requests
- `navimow-banner.svg`: wide release banner for GitHub releases and repository previews
- `navimow-social-card.svg`: social preview graphic for GitHub discussions, release posts, and link cards

Homebridge-specific note:

- runtime plugin UI images still ship from `homebridge-ui/public/`
- the main Homebridge Plugins page icon is managed separately through the Homebridge plugins project
